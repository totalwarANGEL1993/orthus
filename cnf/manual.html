Eine Map bearbeiten

Um eine Map zu bearbeiten, muss sie zuerst geladen werden. Das Programm wird versuchen die ausgewählte Map zu entpacken. Sollte der Vorgang erfolgreich sein, werden die gesperrten Tabs zur weiteren Bearbeitung freigegeben. Die Bearbeitung einer Map ist in 3 Bereiche unterteilt: Grundeinstellungen, Quests und Briefings.

Die Handlung bestimmen

Im Tab "Grundeinstellungen" können einige Einstellungen vorgenommen werden. Es werden Namen, Farben und Beziehungen für Spieler vergeben, die Rohstoffe für den Start eingestellt, Technologien verboten oder erforscht und Optionen des Debug-Mode gesetzt.


Im Tab "Quests erstellen" wird der Ablauf der Handlung erzeugt. Dazu erstellst Du Quests und wählst aus einer Vielzahl von Verhalten. Ein Quest kann über Bedingungen mit anderen Quests in Beziehung gesetzt werden. Bedingungen können sich aber auch auf das Spielgeschehen beziehen. Jeder Quest hat ein oder mehrere Ziele, die der Spieler erfüllen muss. Erreicht der Spieler sein Ziel, wird der Quest erfolgreich abgeschlossen und die eingestellten Belohnungen treten in Kraft. Scheitert der Quest, werden stattdessen die Bestrafungen angewandt.


Da die Handlung einer Map vorwiegend durch Briefings erzählt wird, kannst Du im Tab "Briefings erstellen" eben diese zusammen stellen.

Speichern und laden

Natürlich verlangt niemand von Dir, dass du in einem Rutsch alles fertig stellen kannst. Du kannst, sobald Du eine Map geladen hast, Deinen Fortschritt speichern oder einen bereits gespeicherten Stand in den Editor laden. Dies Geschieht über die Buttons "Speichern" und "Laden". Der Editor legt einige Dateien im JSON-Format an bzw. läd aus diesen Dateien.
