Modules

core.bugfixes This module fixes some minor bugs in the game.
core.oop Small module to create class alike tables that can be instanciated.
core.questbriefing This module adds some briefing improvements.
core.questdebug Implements a debug mode for the quest system featuring visual quest trace on screen, advanced cheats and a simple shell for controlling quests while the map is in development.
core.questsync This module provides a simple synchronization for multiplayer maps.
core.questsystem This is an approach to create a RoaE like quest system.
core.questtools This module provides some helpful tools.
ext.interaction Implements a npc system that can be run seperatly to the vanilla npc system.
ext.onscreeninfo_ex2 This module renders timers and progress bars for quests.
ext.optionsmenu This module allows to create a simple menu controlled by the group selection keys.
ext.speed_ex2 This module allows the player to change the game speed.
ext.troopgenerator This module allows to create AI player and armies.
ext.workplace_ex2 A passive module that allows the player to change the amout of worker in the building.
lib.loader Loads libraries used by the QSB if they are not already present.
multiplayersystem This module aims to create a ruleset for multiplayer for the vanilla game and the History Edition but also be able to use the EMS ruleset from the community server if the map is published there.
questbehavior This is an abstraction layer of the quest system, allowing the user to create quests much more simple.
treasure This module creates better chests.
