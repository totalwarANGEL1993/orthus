Module information_ex2

This module adds some briefing improvements.

Some are just for cosmetics, some are fixing pittful bugs, some offer helpful new features. This is the version for the vanilla game. For Extra 3 we will using mcbBrief by mcb. This will offer at least equal features or maybe even more advanced.

Functions

AddPages (_briefing)
Creates the local briefing functions for adding pages.

Functions created:

  • AP: Creates normal pages and multiple choice pages. You have full control over all settings. It is also possible to do cutscene like camera animations.
  • ASP: Creates a simplyfied page. A short notation good for dialogs. can be used in talkative missions.

Parameters:

  • _briefing table Briefing

Returns:

  1. function AP function
  2. function ASP function
