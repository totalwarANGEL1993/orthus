Module core.questsync

This module provides a simple synchronization for multiplayer maps.

Synchronization is done by creating an event that houses a function and optional parameters. These Events can be fired to synchronize code called from the GUI hence only by one player.

Required modules:

  • qsb.core.oop
  • qsb.lib.questtools
