Module qsb.questbehavior

This is an abstraction layer of the quest system, allowing the user to create quests much more simple.

Also some behavior are recoded and there is a better AI control script for armies than the default controller.

Required modules:

  • oop.lua
  • questsystem.lua

Functions

CreateQuest (_Data)
Creates an quest from the given description.

Description contains of the following entries:

  • Name: Name of quest
  • Receiver: Player that receives the quest
  • Time: Time, until the quest is automatically over
  • Description: Quest information displayed in the quest book
After the fields the behavior constructors are called.

Parameters:

  • _Data [table] Quest description

Returns:

  1. [number] Quest id
  2. [table] Quest instance

Usage:

    CreateQuest {
    Name = "SomeQuestName",
    Description = {
        Title = "Name of quest",
        Text  = "Description of quest",
        Type  = MAINQUEST_OPEN,
        Info  = 1
    },

    Goal_DestroyAllPlayerUnits(2),
    Reward_Victory(),
    Trigger_Time(0)
}
