Module multiplayersystem

This module aims to create a ruleset for multiplayer for the vanilla game and the History Edition but also be able to use the EMS ruleset from the community server if the map is published there.

Just copy the content of MultiplayerRules_Default into your mapscript and rename the table there to MPRuleset_Rules.

Required modules:

  • qsb.lib.oop
  • qsb.core.questsync
  • qsb.questbehavior
  • qsb.ext.optionsmenu
  • qsb.ext.timer
