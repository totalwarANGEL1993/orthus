QuestTools.AreEntitiesOfDiplomacyStateInArea (_player, _position, _range, _state)
Checks the area for entities of other parties with a diplomatic state to the player.

The first 16 player entities in the area will be evaluated. If they're not settler, heroes or buldings, they will be ignored.

Parameters:

  • _player number Player ID
  • _position table Area center
  • _range number Area size
  • _state number Diplomatic state

Returns:

    boolean Entities near
