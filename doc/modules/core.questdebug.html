Module core.questdebug

Implements a debug mode for the quest system featuring visual quest trace on screen, advanced cheats and a simple shell for controlling quests while the map is in development.

Some cheats are disabled in Multiplayer!

Required modules:

  • qsb.core.oop
  • qsb.core.questsystem

QuestSystemDebug

QuestSystemDebug:Activate (_CheckQuests, _DebugKeys, _DebugShell, _QuestTrace) Activates the debug mode.


QuestSystemDebug

QuestSystemDebug:Activate (_CheckQuests, _DebugKeys, _DebugShell, _QuestTrace)
Activates the debug mode. Use the flags to decide which features you want to use.

Parameters:

  • _CheckQuests boolean Call debug check of behavior
  • _DebugKeys boolean Activate debug cheats
  • _DebugShell boolean Activate debug shell
  • _QuestTrace boolean Display quest status changes
