Module ext.speed_ex2

This module allows the player to change the game speed.

If the player should be unable to this feature can be deactivated.

Required modules:

  • qsb.oop
  • qsb.core.questsync
  • qsb.core.questsystem

Methods

GameSpeedSetAllowed (_Flag) Allows or forbid to use speedup.
GameSpeedSetLimit (_Limit) Sets the speed limit for speedup.
