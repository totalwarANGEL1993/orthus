Module treasure

This module creates better chests.

Chests are opened by any hero but you can set a custom condition that is checked first.

By default resources are given as reward. It is also possible to write a custom funktion that will be called after the chest is opened.

Required modules:

  • qsb.oop
  • qsb.questsystem
  • qsb.questbehavior

Methods

SetupTreasureChest (_Data)
Simplified call to create a chest.

Entries in data table:

  • ScriptName: Script name of entity
  • Gold: Amount of gold
  • Clay: Amount of clay
  • Wood: Amount of wood
  • Stone: Amount of stone
  • Iron: Amount of iron
  • Sulfur: Amount of sulfur
  • Condition: Additional opening condition
  • Callback Additional opening callback

Note: Condition and callback are optional. The data of the instance is passed to these function when they are called.

Parameters:

  • _Data table Data table

Returns:

    table Treasure instance
